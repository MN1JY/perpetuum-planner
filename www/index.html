About

Perpetuum Planner is the character planner tool for the Perpetuum sci-fi MMORPG.

Most of the UI elements are copied from in-game dialogs so the credit for UI design goes to Avatar Creations, the makers of Perpetuum. Game data, including names and images, are of course their property too.

Installation

No installation is required. Just save it somewhere and launch. Upgrade is handled by replacing PerperuumPlanner.exe with the new version either manually or via auto-update.
Configuration is stored in %APPDATA%/Perpetuum Planner folder.

The program is confirmed to work on Windows XP SP3, Windows Vista SP2 and Windows 7.
