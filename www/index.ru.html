О программе

Perpetuum Planner - это инструмент планирования развития персонажа для онлайн игры Perpetuum.

Большая часть элементов интерфейса скопирована непосредственно из игры и права на них, а также на все остальные данные заимствованные из игры, принадлежат компании Avatar Creations, создателям Perpetuum.

Установка

Установка программы не требуется. Достаточно скачать и распаковать архив. Обновление программы осуществляется простой заменой файла PerperuumPlanner.exe либо вручную, либо через встроенный механизм автообновления.
Настройки хранятся в папке %APPDATA%/Perpetuum Planner.

Программа рассчитана на работу в среде Windows XP SP3, Windows Vista SP2, Windows 7.

Лицензия

Программа бесплатна и распространяется по принципу "как есть". Автор не несет ответственности за возможную порчу и потерю информации, а также за убытки, понесенные вследствие использования данной программы.

Вы можете свободно распространять эту программу в некоммерческих целях.

Замечания

Приватность: по умолчанию включены регулярные проверки обновлений, вы можете отключить их в настройках.

Данные из игры: данные собраны путем распознавания скриншотов. Пожалуйста, сообщайте о найденных ошибках.
